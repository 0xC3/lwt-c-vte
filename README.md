lwt
===

Lightweight *GTK* and *VTE* based terminal emulator.

Key bindings
------------
The default key bindings of *VTE* are extended with the following:

+   *[ctrl]-[shift]-c*
    -   Copy selection.
+   *[ctrl]-[shift]-v*
    -   Paste selection.
+   *[ctrl]-[shift]-l*
    -   Reset scrollback.
